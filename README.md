# TrackMe
Real time public transport tracking system

## Problem
Many cities lack real-time public transport tracking, causing delays and inconvenience for passengers.

## Solution
TrackMe is a GPS-based system that provides live bus tracking and estimated arrival times.

## Features
- Live bus location tracking
- ETA prediction
- Route monitoring

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask
- Database: MySQL
- Maps: Google Maps API

Prototype / In Progress
